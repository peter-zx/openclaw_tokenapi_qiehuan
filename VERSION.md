v1.2.0 开箱即用版本

【开箱即用】
- 双击 start_with_ui.bat 即可使用
- 自动创建虚拟环境并安装依赖
- 使用阿里云镜像加速下载
- 自动打开浏览器

【功能特性】
- 模型切换：点击卡片快速切换
- 模型通讯录：保存常用模型配置
- 批量导入：支持 TXT 文件/在线导入
- API Key 管理：浏览器本地存储
- 自动重启：切换后自动重启 OpenClaw Gateway
- 跨平台兼容：移除硬编码路径

【使用方式】
1. 下载项目
2. 双击 start_with_ui.bat
3. 自动打开浏览器使用

【系统要求】
- Python 3.10+
- Node.js（首次构建前端）
- 网络连接（下载依赖）
