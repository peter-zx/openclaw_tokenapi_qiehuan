


================================
   OpenClaw 模型切换工具
   开箱即用版本 v1.3.0
================================

一、启动工具
1. 双击文件：start_with_ui.bat
2. 等待自动安装完成（首次运行需要几分钟）
3. 浏览器自动打开，开始使用

二、配置模型
1. 在页面左侧选择服务商（火山引擎/阿里云等）
2. 点击服务商名称右侧的【齿轮图标】
3. 填写配置信息：
   - Provider ID（服务商ID）- 必填
   - Base URL（接口地址）- 必填
   - API Key（密钥）- 必填
   - contextWindow（上下文窗口）- 默认值：64000，范围：1000-128000
   - maxTokens（最大输出长度）- 默认值：8000，范围：100-32000
4. 点击【保存配置】

三、添加模型
1. 在【模型ID】输入框中填写模型名称（如：glm-4-7-251222）
2. 点击【保存到通讯录】保存到列表
3. 点击【保存并应用】立即切换模型

四、切换模型
1. 在右侧找到已保存的模型卡片
2. 直接点击卡片即可快速切换
3. 系统会自动重启 OpenClaw 服务（等待8秒）

================================
注意事项：
- 需要安装 Python 3.10+
- 需要安装 Node.js（仅首次）
- 需要连接网络（下载依赖）
- API Key 只保存在浏览器本地，不会上传
